Manual payments

Payment guide

Use the amount shown on the payment page, then submit only the requested reference or card code.

D17 transfer

  1. Send the exact amount shown on the payment page.
  2. Open the D17 receipt or journal.
  3. Copy only the authorization number.
  4. Paste it into the payment form and submit.

Flouci transfer

  1. Send the exact amount shown on the payment page.
  2. Open the Flouci transaction receipt.
  3. Copy only the transaction ID.
  4. Paste it into the payment form and submit.

TT recharge card

  1. Buy the number of 5 TND TT cards shown on the payment page.
  2. Do not recharge the cards yourself.
  3. Enter every 15-digit card code in the form.
  4. Submit the order for manual review.
Use exactly the number of cards shown on the payment page.

Need help with payment? Contact support before submitting.
